## How to run in your computer

```bash
  # Clone this repository
  git clone https://github.com/tsuyusk/twitter-clone-flexbox

  # Go to 'twitter-clone-flexbox' folder
  cd twitter-clone-flexbox

  # Open 'index.html' file
  start index.html
```
